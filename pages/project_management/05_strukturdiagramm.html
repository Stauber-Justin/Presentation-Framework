Systemübersicht & Modulare Struktur

Das Framework besteht aus folgenden Hauptmodulen:

  • core.js: Einstiegspunkt / Init
  • loader.js: JSON-Slide-Importer
  • slider.js: Navigation & Steuerung
  • ui.js: Fortschritt / Events
  • asset_loader.js: Script & CSS Injection
