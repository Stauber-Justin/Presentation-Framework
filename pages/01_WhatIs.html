Was ist eine SQL-Injection?

Eine SQL-Injection ist ein Angriff, bei dem unerwarteter Code in eine Eingabe eingeschleust wird, die später direkt in einer Datenbank-Abfrage landet.

Beispiel

Nutzer gibt ein:

' OR 1=1 --

und umgeht damit z. B. eine Login-Abfrage:

SELECT * FROM users WHERE username = '' OR 1=1 --';

Ablauf visualisiert:

  • Nutzer gibt manipulierten Text ins Formular ein
  • Formular sendet Eingabe ungeprüft an SQL-Query
  • SQL-Abfrage wird durchbrochen & manipuliert
  • Datenbank gibt alles frei oder führt Schaden aus
„Datenbanken sind ehrlich. Sie führen jeden Befehl aus – selbst wenn es dein Untergang ist.“
